# Consulta CPF Bot 🤖🔍
Este bot automatiza consultas de CPFs via Telegram e armazena os resultados no Google Sheets.

## 📌 Funcionalidades:
- Consulta CPFs automaticamente via Telegram.
- Armazena os dados no Google Sheets.
- Reagenda CPFs sem resposta para nova tentativa.
- Exclui arquivos temporários automaticamente.
- Monitora o progresso do processamento.
